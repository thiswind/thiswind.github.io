目录结构

[ONE]> tree -d
.
├── release
└── src

源代码放在 src 目录当中
编译结果放在 release 目录当中

前提条件

  • src/chinese-gb7714-2005-numeric.csl : 下载
  • src/my-reference.docx : 自己创建一个

模板的创建方法：

  1. 用一个空的 docx 文件作为模板，编译文本
  2. 在编译出来的文本当中，根据需要修改各种样式的字体、字号、段落等
  3. 将修改好的docx文件重命名为my-reference.docx，用作模板

shell test not

2019-11-19

The test command syntax is as follows:

! expression

OR

[ ! expression ]

OR

if test ! condition
then
     command1
     command2
fi
if [ ! condition ]
then
     command1
     command2
fi

如何用`cut`命令找到最后一个字段

2019-11-19 

pwd | rev | cut -d "/" -f 1 | rev

原理：

  1. 逆向
  2. 切分，找到第一项
  3. 逆向，还原
