นี่คือโจทย์การเขียนโปรแกรมแบบสนุกเกี่ยวกับพีชและเมล็ดพีช กติกามีดังนี้:
- เริ่มต้นด้วยจำนวนเงินและเมล็ดพีชในปริมาณหนึ่ง
- ในแต่ละครั้ง คุณสามารถใช้เงิน 1 บาทเพื่อซื้อพีช 1 ลูก หรือใช้เมล็ดพีช 3 เมล็ดแลกพีช 1 ลูก
- เมื่อกินพีช 1 ลูก จะได้รับเมล็ดพีชเพิ่ม 1 เมล็ด

เป้าหมายสุดท้ายคือ: กินพีชให้ได้มากที่สุดเท่าที่จะทำได้ ด้านล่างคือแนวคิดทั้งหมดสำหรับการแก้ปัญหาและโค้ดที่ใช้ได้จริง

## แนวคิดในการแก้ปัญหา

จุดสำคัญของโจทย์นี้คือการใช้คุณค่าของเมล็ดพีชให้คุ้มค่า และหลีกเลี่ยงการเสียทรัพยากรโดยไม่จำเป็น ขั้นตอนโดยรวมมีดังนี้:
1. ในแต่ละขั้น ให้แปลง “มูลค่าของเมล็ดพีช” ให้เป็นเงิน เพื่อให้จำนวนเงินมากที่สุด
2. หากเงินเพียงพอ ให้ซื้อพีช 1 ลูก แล้วกิน
3. เมื่อเริ่มพยายามซื้อแล้วไม่สามารถทำต่อได้ ให้ตรวจสอบว่า “สามารถแลกเพิ่มอีกหนึ่งรอบ” ได้หรือไม่ เพื่อให้การใช้ทรัพยากรมีประสิทธิภาพสูงสุด

## โค้ดโปรแกรม

โค้ดด้านล่างเป็นการทำให้ครบถ้วนด้วย Python:

```python
money = 10
peach_pits = 0
peaches_eaten = 0

while True:
    money += peach_pits / 3
    peach_pits = 0
    
    if money < 1:
        if money + 1/3 >= 1:
            money += 1/3
            peach_pits -= 1
        else:
            break
    
    money -= 1
    peaches_eaten += 1
    peach_pits += 1

print(f"总共吃了 {peaches_eaten} 个桃子")
```
