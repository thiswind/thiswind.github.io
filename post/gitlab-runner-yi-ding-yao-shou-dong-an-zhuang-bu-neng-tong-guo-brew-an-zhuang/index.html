gitlab-runner 一定要手动安装，不能通过 brew 安装

Posted on 2019-11-15，1 min read

否则会下载错误的 gitlab-runner-helper 的docker镜像（已经过期，删掉了，下载不到了）

Download the binary for your system:

sudo curl --output /usr/local/bin/gitlab-runner https://gitlab-runner-downloads.s3.amazonaws.com/latest/binaries/gitlab-runner-darwin-amd64

You can download a binary for every available version as described in Bleeding Edge - download any other tagged release.
Give it permissions to execute:

sudo chmod +x /usr/local/bin/gitlab-runner

https://docs.gitlab.com/runner/install/osx.html#manual-installation-official
