shell test not

2019-11-19

The test command syntax is as follows:

! expression

OR

[ ! expression ]

OR

if test ! condition
then
     command1
     command2
fi
if [ ! condition ]
then
     command1
     command2
fi

如何用`cut`命令找到最后一个字段

2019-11-19 

pwd | rev | cut -d "/" -f 1 | rev

原理：

  1. 逆向
  2. 切分，找到第一项
  3. 逆向，还原
