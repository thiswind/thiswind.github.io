shell test not

发布于 2019-11-19 | 1分钟 | 24字数

The test command syntax is as follows:

! expression

OR

[ ! expression ]

OR

if test ! condition
then
     command1
     command2
fi
if [ ! condition ]
then
     command1
     command2
fi

如何用`cut`命令找到最后一个字段

发布于 2019-11-19 | 1分钟 | 24字数 
pwd | rev | cut -d "/" -f 1 | rev

原理：

  1. 逆向
  2. 切分，找到第一项
  3. 逆向，还原
